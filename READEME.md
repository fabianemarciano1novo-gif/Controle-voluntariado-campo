Controle Voluntariado Campo

Sistema web desenvolvido para otimizar o registro e acompanhamento de visitas realizadas em trabalho voluntário de campo.

*Problema*
Controle manual de 200+ visitas mensais em planilhas dificultava a organização de dados, consulta de histórico e planejamento de rotas.

*Solução*
Aplicação web com formulário digital para registro de visitas, incluindo validação de dados e listagem para análise.

*Tecnologias Utilizadas*
- *Frontend:* HTML5, CSS3, JavaScript
- *Ferramentas:* Git, GitHub, VS Code

*Funcionalidades*
1. Cadastro de nova visita com data, endereço, situação e observações
2. Listagem de visitas registradas na tela
3. Validação de campos obrigatórios
4. Interface simples e responsiva

*Como Executar*
1. Clone o repositório: `git clone https://github.com/[fabianemarciano1novo-gif]/controle-voluntariado-campo.git`
2. Abra o arquivo `index.html` no navegador

*Resultados*
Substituição do controle manual por sistema digital, reduzindo tempo de registro e facilitando a consulta de histórico.

*Aprendizados*
Desenvolvimento web, lógica de programação, versionamento com Git e documentação de projetos.
